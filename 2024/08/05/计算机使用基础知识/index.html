计算机使用基础知识

应邀给新人写的一写注意事项，如果有任何疑问，欢迎给我发邮件询问

一、为什么不能把软件都装在C盘

系统性能：C盘作为系统盘（通常是Windows的主分区），承载了操作系统和许多系统文件。随着使用时间的推移，C盘会积累大量的临时文件、缓存数据以及系统日志等，这会导致磁盘空间逐渐减少，进而影响系统的运行速度和稳定性。此外，当C盘空间不足时，系统可能会频繁进行磁盘碎片整理，进一步降低工作效率。

存储管理：虽然可以通过一些工具或设置来优化C盘的空间使用，但这些方法往往比较复杂且不总是有效。例如，通过磁盘管理合并分区或扩展分区可以释放一些空间，但这需要一定的技术知识，并且操作不当还可能带来风险。另外，有些用户尝试将新软件安装到其他分区以减轻C盘负担，但这一过程通常较为麻烦，需要手动修改注册表和快捷方式等。

权限限制：C盘通常受到严格的权限控制。普通用户没有足够的权限直接修改C盘中的文件和目录，这使得即使想将软件安装到其他分区也面临困难。此外，某些情况下，系统会自动限制对C盘的写入操作，以防止误操作导致系统崩溃。

因此，为了确保计算机的高效运行和数据的安全性，建议将非系统相关的软件和数据安装在其他分区，如D盘或E盘。这样不仅可以避免因C盘空间不足而影响系统性能，还可以简化存储管理和权限设置的工作

二、C盘满了怎么办

第一，可以通过市面上的 C盘清理工具 进行清理，但这大多数是付费的

第二，把不必要的软件都通过特定软件进行迁移，迁到其它盘。你可以给不同的盘不同的定位，比如 D盘 放软件，E盘 放个人文件或者游戏之类的

第三，及时清理你的 回收站和临时文件，通过输入 win+r 打开。

然后输入 %TEMP%，把能删的删了，不能删的跳过

三、我的电脑被 360 入侵了怎么办

众所周知，360 有强大的杀毒功能，同样的，它和病毒一样顽固。如果你的电脑无故多出了 360：
首先，打开快捷方式所在的文件夹，待用
接着，打开控制面板。方法不限，你可以 win+r，然后输入 control 回车
然后，选择卸载程序，搜索360，把有关文件都卸载

接着卸载360卫士，不管它怎么说，都选卸载
最后，在 C:\Program Files (x86)\360\360Safe 中，把残存的文件都删除，如果提示被占用，首先检查是否已经把它彻底退出。如果已经彻底退出，可以使用火绒，火绒支持解除占用的功能

四、我的电脑太长时间不关机卡了

这是正常现象，长期处于开机状态会导致许多进程在后台运行。如果不能保证自己能及时手动关闭这些进程，请定时关机（一天一次或者两天一次）
同样，太频繁的开关机也会损害电脑使用寿命（它可能会蓝屏：( 给你抗议一下）

五、刚下载的文件怎么不见了

如果你是在浏览器下载，那么像下图一样打开即可，下载好的文件后面会有个文件夹图标，点开即可看到它的位置。
如果你没有进行额外的配置的话，它大概率会在 C盘的Downloads下，你可以直接打开看看它在不在

六、为什么我的文件看不到后缀

这可能是新手常见的问题，这里我以 win11 为例，展示怎么显示拓展名，首先打开任意一个文件夹，接着，跟着图中标注的顺序进行点击即可

七、这个后缀的文件我没见过打不开怎么办

这种情况其实并不少见，很多时候我们需要借助搜索引擎来解决问题
这个时候，打开你的 Bing / Google / Firefox / 实在没有百度凑活一下也行，输入“xxx后缀 怎么打开” 即可，这个时候可能会给你推荐 某种解析器/阅读器 之类的，你唯一需要做的，就是再查一查它正不正经，会不会给你的电脑带来不可预估的风险（如果可以，找正版或者有官方许可的软件），然后把它下载下来

八、我不希望我的电脑中毒

性价比最高的：火绒，不要钱，能杀掉市面上大部分木马
此外，如果你是初学者，我并不建议你把 windows defender 关了（默认是开启的），尽管它有时候可能会对你正常运行的软件痛下杀手（极小概率），但相信我，这是为了保护你的电脑，请给它一些容忍
